### 算法框架

回溯算法总结起来就是一个**多叉树的遍历问题**,关键就是在前序遍历和后序遍历的位置做一些操作,算法框架:

```
result = []
void backtrack(路径,选择列表) {
    if(满足结束条件) {
        result.add(路径)
        return
    }
    for(选择 in 选择列表) {
        做选择
        backtrack(路径,选择列表)
        撤销选择
    }
}
```

写backtrack函数时,需要维护走过的"路径"和当前可以做的"选择列表",当触发"结束条件"时,将路径计入结果集.
